# Frost Factory - Premium Cake Shop

A modern, responsive website for Frost Factory, a premium cake shop. Built with HTML5, CSS3, and vanilla JavaScript, featuring a sleek design with smooth animations and interactive components.

## 🍰 Features

- **Responsive Design** - Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Navigation Bar** - Fixed navigation with scroll effect and active link highlighting
- **Hero Section** - Eye-catching hero section with overlay and call-to-action
- **About Section** - Information about Frost Factory
- **Menu Display** - Showcase of cake offerings and products
- **Testimonials** - Customer reviews and feedback section
- **Team Section** - Meet the team behind Frost Factory
- **Contact Section** - Easy ways to get in touch
- **Smooth Animations** - CSS transitions and JavaScript interactions for enhanced user experience
- **Modern UI/UX** - Beautiful gradient effects and professional styling

## 🛠️ Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Advanced styling with gradients, animations, and flexbox/grid layout
- **JavaScript** - DOM manipulation and interactive features
- **Bootstrap 5.3.2** - Responsive grid system and components
- **Font Awesome 6.4.2** - Icon library for visual enhancements
- **Google Fonts** - Playfair Display and Poppins typography

## 📁 Project Structure

```
Cake_Shop/
├── index.html          # Main HTML file with all sections
├── style.css           # Custom CSS styles and animations
├── script.js           # JavaScript for interactivity
├── img/                # Image assets folder
└── README.md          # Project documentation
```

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional installations required - this is a static website

### Installation

1. Clone or download the project folder
2. Open `index.html` in your web browser
3. The website will load with all styling and functionality

### Running the Project

Simply open the `index.html` file in any web browser:
```bash
# Windows
start index.html

# Mac
open index.html

# Linux
xdg-open index.html
```

## 📋 Usage

### Sections

- **Home** - Landing section with hero image and primary message
- **About** - Company background and story
- **Menu** - Product showcase and offerings
- **Testimonials** - Customer reviews and ratings
- **Our Team** - Staff profiles and information
- **Contact** - Contact information and inquiry form

### Navigation

- Click on navbar links to scroll to different sections
- The navbar includes a fixed position for easy access
- Active link highlighting shows your current section
- Navbar styling changes on scroll for better UX
